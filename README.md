# 🚗 CarShowroom

> SPA-додаток для перегляду автомобільного каталогу з детальними сторінками та відгуками.

![TypeScript](https://img.shields.io/badge/TypeScript-5-3178C6?style=flat-square&logo=typescript)

## 📌 Про проект

**CarShowroom** — це тестове завдання у форматі Single Page Application з двома основними розділами:

- **Каталог** — перегляд автомобілів з фільтрацією за параметрами
- **Деталі авто** — повна інформація про обраний автомобіль та відгуки користувачів

## 🛠 Стек технологій

| Технологія | Призначення |
|---|---|
| React | UI-бібліотека |
| TypeScript | Статична типізація |
| Vite | Збірка та dev-сервер |
| React Router | Клієнтська маршрутизація |

## 🚀 Запуск проекту

```bash
# Клонуємо репозиторій
git clone https://github.com/Nickkklll/CarShowRoom.git
cd car-showroom

# Встановлюємо залежності
npm install

# Запускаємо dev-сервер
npm run dev
```

Додаток буде доступний за адресою: `http://localhost:5173`

## 📁 Структура проекту

```
src/
```
